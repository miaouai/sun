# 🌞 向日窥 - Sun Peek

**阳台光照智能分析应用**

---

## 📖 项目概述

### 定位
通过手机朝向检测 + 地理位置信息，智能分析阳台日照情况的专业工具。

### 核心价值
用户只需站在阳台上打开 App，系统自动计算：
- 🌅 当地日出/日落时间
- ☀️ 有效日照时长和具体时间段
- 📊 实时光照强度预估

---

## 🎯 功能特性

### 1️⃣ 权限管理模块
- ✅ 页面加载时自动请求定位权限
- ✅ iOS 13+ 方向传感器权限适配
- ✅ 权限状态实时监控与 UI 反馈

### 2️⃣ 方向检测模块
- **小指南针可视化** - 四大方位标注度数
- **自动检测模式** - 实时动态更新朝向数据
- **手动输入模式** - 精确设置特定角度值
- ⚔️ 两种模式互斥切换，灵活可控

### 3️⃣ 阳台配置模块
基于科学建筑学数据的光照影响模型：
- **凸出式阳台** - 三面采光，光照利用率 +35%
- **内嵌式阳台** - 单面采光，需考虑遮挡系数

### 4️⃣ 位置信息模块
- 🛰️ GPS 高精度定位（优先）
- 🌐 IP 地理定位（降级方案）
- 🔍 反向地理编码识别城市
- 🇨🇳 支持大陆/港澳台地区手动选择

### 5️⃣ 日照分析引擎
综合天文学算法输出：
- 🧭 当前朝向（东南西北精确度）
- ⏱️ 今日有效日照时长（分钟级精度）
- 🌄 日出/日落时刻
- 🏢 墙面遮挡影响评估
- 📈 季节变化趋势预测

## 🛠️ 技术架构

### 前端技术栈
- **HTML5** - 语义化结构
- **CSS3** - 响应式设计 + 动画效果
- **原生 JavaScript** - 无框架依赖，轻量高效

### 核心 API
| API | 用途 | 兼容性 |
|-----|------|--------|
| Geolocation API | GPS 定位 | 95%+ 移动端 |
| DeviceOrientation API | 方向检测 | iOS 13+/Android |
| Permissions API | 权限预检 | 现代浏览器 |
| GeoCoder (IP) | IP 转地址 | 备用方案 |

### 天文计算库
- **SunCalc** - 日出日落时间计算
- **自建算法** - 方位角与光照时长模型
